# SB Admin 2 Frontend

A modern admin dashboard template built with HTML, CSS, and JavaScript. This project is a frontend implementation of the popular SB Admin 2 template.

## Features

- Responsive design
- Modern UI components
- Interactive charts and tables
- User authentication pages
- Profile management
- Activity logging

## Pages

- Dashboard (`/`)
- Login (`/login.html`)
- Register (`/register.html`)
- Forgot Password (`/forgot-password.html`)

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Bootstrap 4
- Chart.js
- jQuery

## Local Development

1. Clone the repository:
```bash
git clone https://github.com/vitor-dornela/node-DevWeb2-c1.git
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

The application will be available at `http://localhost:3000`
